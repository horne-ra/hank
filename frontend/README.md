# Hank frontend

Next.js App Router UI for Hank. Local setup and architecture are documented in the [**repository root README**](../README.md).

Quick reference:

- **With the rest of the stack:** from the repo root, use `./dev.sh` or `docker compose up --build` (see root README).
- **Frontend only (expects the Python backend on port 8000):** `pnpm dev` in this directory.
